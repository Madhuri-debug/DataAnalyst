# Data Analyst Portfolio

## Project 1: Global Sales Analysis Dashboard

An interactive Power BI dashboard built for analyzing global sales performance across multiple countries and product categories.

### Tools Used

* Power BI
* Data Visualization

### Dashboard Features

* Total Revenue KPI
* Country-wise Sales Comparison
* Monthly Revenue Trend
* Top Product Categories
* Interactive Country Filter

## Business Insights

This dashboard helps identify:

* High-performing countries
* Sales trends over time
* Best-selling product categories
## Project 2: Employee Salary Analysis using SQL

SQL-based analysis project for evaluating employee salary trends and departmental performance.

### Tools Used

* SQL Server
* SQL Queries
* Data Analysis

### Concepts Used

* GROUP BY
* HAVING
* Subqueries
* Window Functions
* Ranking Functions

### Business Questions Solved

* Which department has the highest average salary?
* Which employees earn above company average?
* Who is the top-paid employee in each department?

### Key Insights

This project helps analyze salary distribution and identify department-wise compensation patterns.
